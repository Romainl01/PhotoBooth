# Morpheo - AI Photobooth

Morpheo is an AI-powered creative photo transformation web application that instantly transforms user photos into themed visual styles using Google's Gemini AI.

## Quick Start

1.  Clone the repository.
2.  Navigate to `nextjs-app`.
3.  Install dependencies: `npm install`.
4.  Create `.env.local` with your `GOOGLE_API_KEY`.
5.  Run development server: `npm run dev`.

See [Getting Started](docs/03_GETTING_STARTED.md) for detailed instructions.
